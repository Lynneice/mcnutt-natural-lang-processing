# Text Mining Models

This directory contains text mining models and related resources.

## Structure

- **models/** - Trained mining models (topic models, keyword extraction, etc.)
- **data/** - Training and test datasets for text mining
- **notebooks/** - Jupyter notebooks for exploration and experimentation
- **scripts/** - Training, evaluation, and inference scripts

## Usage

Place your text mining model files, datasets, and code in the appropriate subdirectories.
