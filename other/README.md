# Other NLU Models

This directory contains other NLU models that don't fit into the main categories.

## Structure

- **models/** - Trained models (NER, entity extraction, classification, etc.)
- **data/** - Training and test datasets
- **notebooks/** - Jupyter notebooks for exploration and experimentation
- **scripts/** - Training, evaluation, and inference scripts

## Usage

Place your NLU model files, datasets, and code in the appropriate subdirectories.
Examples: Named Entity Recognition, Entity Extraction, Custom Classification, etc.
